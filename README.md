# CRED Clone — Premium Fintech Landing Page

A modern premium fintech landing page inspired by CRED, built using React, Vite, Tailwind CSS v4, and Framer Motion.

---

# ✨ Features

- Premium dark UI
- Responsive design
- Smooth animations
- Glassmorphism cards
- NeoPOP-inspired sections
- Interactive hover effects
- Animated counters
- Tailwind CSS v4
- Framer Motion animations
- Vite fast build setup

---

# 🛠 Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Framer Motion
- GS
